Installation

1. Download the latest version of CMOTD

2. Drop it into the plugins folder

3. Restart your BungeeCord

4. Edit the configuration (plugins/CMOTD/config.yml)

5. Done

Commands

Command Description Default-Permission
/cmotd Displays info message -
/cmotd reload Reload the configuration cmotd.reload
/cmotd help Displays help message cmotd.help
/maintenance Displays info message -
/maintenance on Turn on the maintenance mode cmotd.maintenance.on
/maintenance off Turn off the maintenance mode cmotd.maintenance.off
/maintenance help Displays help message cmotd.maintenance.help
